# Personal Technical Portfolio

Selamat datang di repositori portofolio web saya. Website ini dirancang sebagai halaman tunggal (*single-page portfolio*) yang responsif untuk menampilkan profil singkat, keahlian teknis, pengalaman akademik, serta beberapa proyek pengembangan perangkat lunak (*software development*) yang telah saya kerjakan.

Website ini dikembangkan menggunakan **HTML** dan dikustomisasi menggunakan utility-first CSS framework **Tailwind CSS**.

---

## Tech Stack & Fitur Web

- **Dasar Web:** HTML5 & Semantics Element
- **Styling:** Tailwind CSS (via CDN)
- **Desain:** Minimalis, Modern, dan Responsif (Mobile & Desktop Friendly)
- **Deployment Platform:** Railway

---

## Proyek Pilihan yang Ditampilkan

Di dalam portofolio ini, saya menonjolkan 3 proyek utama yang mencakup siklus SDLC (pengembangan hingga pengujian):
1. **Mommy Bakery (Web UMKM):** Berfokus pada Backend Engineering awal, integrasi API OTP WhatsApp, pengkondisian *error handling*, dan pengelolaan *deployment*.
2. **CompuPay (Sistem Manajemen Karyawan):** Berfokus pada logika backend (*Point Record*), agregasi data, dan penyediaan infrastruktur REST API yang diuji menggunakan Postman.
3. **WebAdmin (Panel Kontrol Dashboard):** Berfokus pada Frontend Engineering, konsumsi API, dan integrasi UI/UX yang responsif.
